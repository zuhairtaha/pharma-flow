# فارما فلو — نظام إدارة الصيدلية

تطبيق ويب بالعربية لإدارة الصيدليات (مخزون، موردون، عملاء، فواتير مبيعات). مبني بأحدث تقنيات الفرونت إند:

- **React 19** (`createRoot`, `use()`, الـ JSX الجديد)
- **React Router v7** (`HashRouter`, `NavLink`, nested routes)
- **Tailwind CSS v4** (عبر `@tailwindcss/vite` + `@theme` directive)
- **Vite 6** لبناء التطوير السريع

تصميم مبني على مبادئ **Material Design 3** (ألوان، شكل مستدير، ارتفاعات/Elevation، ripple بسيط، navigation drawer قابل للطي)، **RTL** افتراضي، و**Responsive** من الهاتف حتى شاشة سطح المكتب.

## التشغيل

```bash
npm install
npm run dev
```

ثم افتح المتصفح على `http://localhost:5173`.

## قاعدة البيانات

التطبيق يستخدم **ملف JSON داخل `localStorage`** بدون أي backend. يمكن تصديره كـ ملف `.json` من شاشة الإعدادات، واستيراده لاحقاً.
